Quickstart

A complete script in one screen. Find a key in the heap, mutate it, draw on the host's frame, exit cleanly.

A first script

Open the script panel, paste the snippet below, press Run. This script prints the size of the heap and exits.

hello.lua 
1local tables = getgc_count("table")
2local strings = getgc_count("string")
3
4print("heap: " .. tables .. " tables, " .. strings .. " strings")

Patch a value

gctab finds tables that contain a given string key and writes new values into the matching slots.

patch.lua 
1local cache = gctab.find({ "WalkSpeed", "JumpHeight" })
2
3local n = gctab.set(cache, {
4    WalkSpeed  = 100,
5    JumpHeight = 200,
6})
7
8print("patched " .. n .. " slot(s)")

Re-running this script after a respawn or rejoin works without re-walking the heap. Call gctab.refresh(cache) to drop stale entries and pick up new tables.

Draw on the host frame

Create a persistent shape, mutate its properties from a render loop, remove it when you're done.

crosshair.lua 
1local w, h = viewport_size()
2local cx, cy = w * 0.5, h * 0.5
3
4local dot = Drawing.new("Circle")
5dot.Position = Vector2.new(cx, cy)
6dot.Radius   = 2
7dot.Color    = Color3.fromRGB(232, 155, 111)
8dot.Filled   = true
9dot.Visible  = true
Drawings persist across script runs

Call :Remove() on the handle when you're finished. Otherwise the dot will continue rendering until the runtime exits.

Putting it together

A full example covering all three primitives. Save as aim.lua:

aim.lua 
 1local Players = game:GetService("Players")
 2local Local   = Players.LocalPlayer
 3
 4local reticle = Drawing.new("Circle")
 5reticle.Radius  = 3
 6reticle.Filled  = true
 7reticle.Color   = Color3.fromRGB(232, 155, 111)
 8reticle.Visible = true
 9
10local function closest()
11    local mx, my = getmousepos()
12    local best, dist = nil, math.huge
13
14    for _, p in ipairs(Players:GetPlayers()) do
15        if p ~= Local and p.Character then
16            local hrp = p.Character:FindFirstChild("HumanoidRootPart")
17            if hrp then
18                local sx, sy, on = world_to_screen(hrp.Position)
19                if on then
20                    local dx, dy = sx - mx, sy - my
21                    local d = (dx * dx + dy * dy) ^ 0.5
22                    if d < dist then best, dist = p, d end
23                end
24            end
25        end
26    end
27    return best
28end
29
30while true do
31    local mx, my = getmousepos()
32    reticle.Position = Vector2.new(mx, my)
33    local target = closest()
34    reticle.Color = target
35        and Color3.fromRGB(232, 155, 111)
36        or  Color3.fromRGB(120, 120, 120)
37    wait(1 / 120)
38end

Paints a small dot under the mouse and tints it amber whenever there's a target visible on screen. Closing the script panel removes the drawing automatically when the runtime tears down.
