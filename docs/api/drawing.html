Drawing

Persistent shapes drawn on the host's swap chain. Every shape is a handle. Mutate properties to update; call :Remove() to free.

module

Drawing.new

Constructs a new shape of the given kind.

Drawing.new(kind: string) → Drawing
KindShapeNotes
Lineline segmentFrom / To, Thickness
Squareaxis-aligned rectPosition, Size, Filled
CirclecirclePosition, Radius, NumSides, Filled
TextlabelPosition, Text, Font, Size, Center
Trianglethree pointsPointA, PointB, PointC, Filled
Quadfour pointsPointA..D, Filled

Common properties

PropertyTypeDescription
VisiblebooleanShows or hides the shape.
ZIndexnumberRender order. Higher draws over lower.
ColorColor3Fill or stroke color.
Transparencynumber0 opaque, 1 invisible.
ThicknessnumberStroke width when applicable.
FilledbooleanSolid fill vs outline only.
OutlinebooleanAdds a contrasting outline.

Example

lua 
1local dot = Drawing.new("Circle")
2dot.Position = Vector2.new(200, 200)
3dot.Radius   = 3
4dot.Color    = Color3.fromRGB(255, 111, 177)
5dot.Filled   = true
6dot.Visible  = true

Lifecycle

Shapes persist across script runs unless you remove them. Always pair Drawing.new with a :Remove() in your teardown path.
