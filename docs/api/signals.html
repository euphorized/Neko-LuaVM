Signals

RBXScriptSignal and RBXScriptConnection. Connect callbacks to game ticks, instance events, and input.

module

Basics

Every signal exposes Connect and Once. Both return an RBXScriptConnection with a Connected field and a Disconnect method.

signal:Connect(callback: function) → RBXScriptConnection signal:Once(callback: function) → RBXScriptConnection conn.Connected → boolean conn:Disconnect() → ()

Run-loop signals

Always reachable, no instance lookup needed. Each callback receives the delta time in seconds since the previous tick.

RunService.Heartbeat:Connect(function(dt) end) RunService.PostSimulation:Connect(function(dt) end) RunService.Stepped:Connect(function(dt) end) RunService.PreSimulation:Connect(function(dt) end) RunService.RenderStepped:Connect(function(dt) end) RunService.PreRender:Connect(function(dt) end)

Heartbeat and PostSimulation are aliases for the same source. Stepped and PreSimulation are aliases. RenderStepped and PreRender are aliases.

Example

lua 
1local rs = game:GetService("RunService")
2local n = 0
3local conn = rs.Heartbeat:Connect(function(dt)
4    n = n + 1
5    if n >= 120 then conn:Disconnect() end
6end)

Instance events

Indexing an event name on any instance returns a dynamic RBXScriptSignal. The signal is polled each heartbeat against a baseline taken at first Connect.

inst.ChildAdded:Connect(function(child) end) inst.ChildRemoved:Connect(function(child) end) inst.DescendantAdded:Connect(function(child) end) inst.DescendantRemoving:Connect(function(child) end) inst.AncestryChanged:Connect(function(self, parent) end) basepart.Touched:Connect(function(other) end) basepart.TouchEnded:Connect(function(other) end) humanoid.StateChanged:Connect(function(old, new) end) humanoid.Died:Connect(function() end)

Touched and TouchEnded are AABB-overlap polled. They will not match physics-engine touches that resolve to non-overlapping geometry.

Input service signals

Backed by the live UserInputService hook so the latency matches the host. The input table mirrors the standard InputObject shape.

UserInputService.InputBegan:Connect(function(input, processed) end) UserInputService.InputEnded:Connect(function(input, processed) end) UserInputService.InputChanged:Connect(function(input, processed) end)

Example

lua 
1local uis = game:GetService("UserInputService")
2uis.InputBegan:Connect(function(input, processed)
3    if processed then return end
4    if input.KeyCode == Enum.KeyCode.E then
5        print("E pressed")
6    end
7end)

RBXScriptConnection

The handle returned by Connect and Once. Holding a connection alive holds the registered callback alive too. Calling Disconnect drops both.

conn.Connected → boolean conn:Disconnect() → () conn:disconnect() → ()
