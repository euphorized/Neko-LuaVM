Instance

Live tree access. Backed by direct memory reads against the host's instance graph.

module

Instance.new

Constructs a new instance of the given class. The handle is returned immediately. Property writes are queued and committed when the instance gets a parent.

Instance.new(className: string) → Instance

Example

lua 
1local p = Instance.new("Part")
2p.Name        = "marker"
3p.Anchored    = true
4p.Color       = Color3.fromRGB(255, 111, 177)
5p.Size        = Vector3.new(2, 2, 2)
6p.Parent      = workspace

Services

Top-level services live under game. Always resolve them through GetService rather than FindFirstChild.

game:GetService(name: string) → Instance game:FindService(name: string) → Instance?

Traversal

All standard tree walkers are present.

inst:FindFirstChild(name: string) → Instance? inst:WaitForChild(name: string) → Instance inst:FindFirstChildOfClass(class: string) → Instance? inst:FindFirstChildWhichIsA(class: string) → Instance? inst:FindFirstAncestor(name: string) → Instance? inst:GetChildren() → { Instance } inst:GetDescendants() → { Instance }

Identity

inst:IsA(class: string) → boolean inst:GetFullName() → string

Lifetime

inst:Clone() → Instance inst:Destroy() → () inst:Remove() → ()

Destroy and Remove are aliased and behave identically: parents the instance to nil and locks the parent property.

Players service

The Players service exposes the standard roster.

Players:GetPlayers() → { Player } Players.LocalPlayer → Player

HttpGet

Synchronous HTTP fetch. Returns the response body as a string. Throws on non-2xx.

inst:HttpGet(url: string) → string inst:HttpGetAsync(url: string) → string
