Globals

Top-level functions visible from any script. Mostly Roblox-compatible — print, typeof, task behave the way scripts expect — with a few extras for working with the host.

global

I/O

print(...)

Forwards all arguments to the runtime console. Tab-separated, newline-terminated, no formatting.

print(...: any) → ()

warn(...)

Same as print but renders the line in the warning color.

setclipboard(s) / toclipboard(s)

Replaces the system clipboard contents with the given string. Returns true on success.

setclipboard(text: string) → boolean

Time and tasks

tick()

Seconds since process start, monotonic, double-precision.

tick() → number

wait(seconds)

Suspends the current Lua thread for at least seconds. The runtime continues to pump the host bridge during the wait — drawings keep rendering, signals keep firing.

wait(seconds: number?) → number

Returns the actual elapsed time. The minimum resolution is roughly one frame.

task

A scheduler with five entry points. All execute on the same VM thread — there is no parallelism.

task.spawn(fn: function) → () task.defer(fn: function) → () task.delay(seconds: number, fn: function) → () task.wait(seconds: number?) → number task.tick() → number

task.spawn calls the function synchronously. task.defer queues it for the end of the current tick. task.delay queues it after a given duration. Errors thrown by deferred or delayed callbacks surface in the console with a full traceback.

loadstring(src, chunkname?)

Compiles a Luau source string and returns a callable function. On compile error returns nil, err.

loadstring(src: string, chunkname: string?) → function | (nil, string)

Input probing

iskeydown(vk)

Returns true if the given virtual key is currently down. Uses GetAsyncKeyState — no event filtering, no double-press detection.

iskeydown(vk: number) → boolean

ismousedown(button?)

Same, for mouse buttons. 1 = left, 2 = right, 3 = middle. Defaults to left.

getmousepos()

Returns the cursor's screen position in pixels.

getmousepos() → number, number

Display

viewport_size()

Width and height of the host's render target, in pixels. Updates on resize without re-querying.

viewport_size() → number, number

world_to_screen(v)

Projects a world-space Vector3 into screen pixel coordinates using the host's cached view matrix.

world_to_screen(point: Vector3) → number, number, boolean

Returns three values: x, y, and a boolean indicating whether the point is in front of the camera and within the viewport. When the boolean is false, the coordinates are -1, -1.

Introspection

typeof(v)

Returns the type name of v as a string. Knows about all the built-in types — "Vector3", "CFrame", "Instance", etc. Falls back to Lua's type for primitives.

typeof(value: any) → string
