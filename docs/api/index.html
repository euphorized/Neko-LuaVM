Conventions

Every function in this reference is documented with the same shape: a one-line summary, a signature, a parameter table, the return value, and at least one runnable example.

Where a function is also reachable as a method on an object, both forms are noted.

Searching

Press ⌘K (or Ctrl+K on Windows) to open the command palette. It indexes every page, every reference, and every public function — including subpaths like gctab.find and Drawing.new.
