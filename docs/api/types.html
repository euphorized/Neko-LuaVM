Types

Built-in value types. All immutable, all comparable, all carry a stable typeof name.

core

Vector3

Three-component float vector. Supports arithmetic, dot/cross products, magnitude, unit vector.

Vector3.new(x: number, y: number, z: number) → Vector3
FieldTypeDescription
X, Y, ZnumberComponents.
MagnitudenumberLength of the vector.
UnitVector3Normalized direction.

Constants: Vector3.zero, Vector3.one, Vector3.xAxis, Vector3.yAxis, Vector3.zAxis.

Vector2

Two-component float vector. Same surface as Vector3 minus Z.

Vector2.new(x: number, y: number) → Vector2

CFrame

3D coordinate frame: position plus orientation matrix. Used for parts, cameras, anything spatial.

CFrame.new() → CFrame CFrame.new(x, y, z) → CFrame

Color3

RGB color in normalized 0-1 floats.

Color3.new(r, g, b) → Color3 Color3.fromRGB(r, g, b) → Color3

UDim and UDim2

Scale + offset coordinate types used by GUI sizes.

UDim.new(scale, offset) → UDim UDim2.new(xScale, xOffset, yScale, yOffset) → UDim2 UDim2.fromOffset(x, y) → UDim2

NumberRange

Closed numeric interval.

NumberRange.new(min, max) → NumberRange

Rect

Axis-aligned rectangle in screen space.

Rect.new(minX, minY, maxX, maxY) → Rect
