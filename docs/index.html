API surface

Every module is reachable from a vanilla Luau script through globals or library tables. There is no hidden state, no implicit context, no metaprogramming layer between the script and the engine.

globals print, warn, typeof, tick, task scheduler, clipboard, world-to-screen. core types Vector3, Vector2, CFrame, Color3, UDim, UDim2, NumberRange, Rect. core Instance Live tree access. FindFirstChild, GetChildren, GetService, IsA. module Drawing Persistent shapes drawn on the host's swap chain. module gctab Find live tables by key name. Mutate value slots in place. module

Philosophy

Boring under load. The VM never crashes the host. Every cross-process read is bounded, validated, and recoverable. Stale references degrade silently rather than fault.

Self-describing surface. If you can call it from Lua, you can read its source. typeof(x) tells you what you have.

Symmetry between read and write. Anything you can observe through getgc or instance reflection, you can mutate through the same primitives. Validation rules are identical in both directions.

External by design

This runtime executes in a separate process from the host. Pointers returned from getgc identify host-side objects. They cannot be dereferenced as Luau values inside the VM.
